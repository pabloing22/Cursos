Desde https://babeljs.io/docs/en/usage instalamos babel

Ejecutamos:
npm install --save-dev @babel/core @babel/cli @babel/preset-env
core: Paquete del core de babel
cli: Comando para usar babel
Preset: colección de plugins que nos permite tomar el código ES6 y lo transforma a JS

Antes de instalar necesitamos crear el package.json que nos va a servir para nodeJS
